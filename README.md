## Vision

Segment, extract properties and classify images

### Setup

Install Anaconda 3-5 via Py Env:

```zsh
pyenv install anaconda3-5.2.0
```

Set Anaconda to local project:

```zsh
pyenv local anaconda3-5.2.0
```

### Run

```zsh
python script.py
```
