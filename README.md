#### Dynavision

> :house: Computer vision in Python 3

#### Setup

To run install [pyenv](https://github.com/pyenv/pyenv) and Anaconda

```shell
pyenv install anaconda3-5.2.0
```

Set the Anaconda to global or local scope

```shell
pyenv global anaconda3-5.2.0
```

#### Run

```shell
python l00_e00.py
```
